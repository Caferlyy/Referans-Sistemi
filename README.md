 | VERİTABANI | 
```

    {
        "driver"    "sqlite"
        "host"        "localhost"
        "database"    "referal_system-sqlite"
        "user"        ""
        "pass"        ""
        //"timeout"    "0"
        "port"        "3306"
    }
 ```
